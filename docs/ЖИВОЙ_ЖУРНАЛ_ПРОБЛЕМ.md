## ✅ Решенные проблемы

### [2025-10-16] Проблема #1: Адаптивность карточек транзакций

**Описание:**  
На мобильных устройствах карточки транзакций выглядели перегруженными из-за горизонтального layout.

**Причина:**  
Использовался flex-row для всех размеров экрана, что сжимало контент на маленьких экранах.

**Решение:**
```tsx
// Было
<div className="flex flex-row items-center justify-between">

// Стало
<div className="flex flex-col sm:flex-row sm:items-center sm:justify-between gap-4">
```

**Результат:**  
✅ На mobile: вертикальный layout (flex-col)  
✅ На desktop: горизонтальный layout (flex-row)  
✅ Контент читаемый на всех экранах

---

### [2025-10-16] Проблема #2: Производительность анимаций

**Описание:**  
Множество Framer Motion анимаций замедляли рендеринг на слабых устройствах.

**Причина:**  
Использование Framer Motion для простых анимаций (hover, fade) было избыточным.

**Решение:**
```css
/* Простые анимации через CSS */
.transition-smooth {
  transition: all 0.3s cubic-bezier(0.4, 0, 0.2, 1);
}

/* Framer Motion только для сложных */
<motion.div
  animate={{ x: [0, 100, 0] }}  // Сложная анимация
>
```

**Результат:**  
✅ CSS для hover/transitions (быстрее)  
✅ Framer Motion для сложных (orbs, stagger)  
✅ 60 FPS на большинстве устройств

---

### [2025-10-16] Проблема #3: Размер шрифтов на мобильных

**Описание:**  
Заголовки и текст были слишком большими на маленьких экранах.

**Причина:**  
Фиксированные размеры шрифтов без адаптации.

**Решение:**
```tsx
// Было
<h2 className="text-5xl font-bold">

// Стало
<h2 className="text-3xl sm:text-5xl font-bold">
```

**Результат:**  
✅ Mobile: text-3xl (30px)  
✅ Desktop: text-5xl (48px)  
✅ Читаемость улучшена

---

### [2025-10-16] Проблема #4: Скрытие navigation labels

**Описание:**  
Кнопка "Подключить кошелёк" была слишком длинной на мобильных.

**Причина:**  
Полный текст не помещался в кнопку.

**Решение:**
```tsx
<span className="hidden sm:inline">Подключить кошелёк</span>
<span className="sm:hidden">Войти</span>
```

**Результат:**  
✅ Mobile: короткий текст "Войти"  
✅ Desktop: полный текст "Подключить кошелёк"  
✅ UI не сломан
