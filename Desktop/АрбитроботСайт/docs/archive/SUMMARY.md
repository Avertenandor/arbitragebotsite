## 🚀 Как запустить

### Quick Start

```bash
# 1. Development
npm install
npm run dev
# → http://localhost:3000

# 2. Production
npm run build
npm run start
# → http://localhost:3000

# 3. Deploy (Vercel)
vercel --prod
# → https://your-domain.vercel.app
```

### Environment Setup

```bash
# .env.local
NEXT_PUBLIC_BSC_RPC_URL=https://bsc-dataseed.binance.org/
NEXT_PUBLIC_PLEX_ADDRESS=0x...
NEXT_PUBLIC_AUTH_ADDRESS=0x...
```
